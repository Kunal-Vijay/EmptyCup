# EmptyCup

# Installation

1. Clone the repository
```sh
git clone git@github.com:Kunal-Vijay/EmptyCup.git
```

2. Install dependencies
```sh
 npm i 
```
3. Run server
```sh
npm start
```
4. Run frontend
```
npm run start
```
# Environment Variables
```sh
REACT_APP_BASE_URL
MONGO_URL
```
